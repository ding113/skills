name: translate-article
description: Translate articles between Chinese and English while preserving structure, terminology, tone, and formatting — Markdown, links, and code blocks stay untouched. Use when the user wants an article, blog post, or doc translated, localized, or turned bilingual.
---

> **Draft** — scope agreed, behavior not yet written. Do not rely on this skill yet.

The planned shape:

1. **Read before translating** — extract the article's register, audience, and recurring terminology; build a small glossary first so terms stay consistent throughout.
2. **Translate by meaning, not by sentence** — natural phrasing in the target language over word-for-word fidelity; no translationese.
3. **Formatting invariants** — Markdown structure, links, anchors, code blocks, and inline code are preserved exactly; only prose is translated.
4. **Final pass** — re-read the output alone, without the source, and fix anything that still reads like a translation.
