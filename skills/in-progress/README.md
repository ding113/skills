# In Progress

Drafts not yet ready to ship: real frontmatter, unfinished bodies. Expect rough edges and breaking changes. They're excluded from the plugin and the top-level README reference until they graduate to a promoted bucket.

- **[article-writing](./article-writing/SKILL.md)** — Publication-quality article writing in Simplified Chinese and English across three genres (blog, techdoc, publication): discovery brief → genre-aware drafting → compiled rule bundles + lint + web fact-checking → rewrite loops until zero findings. Graduates to `writing/`.
- **[get-pr-comments](./get-pr-comments/SKILL.md)** — Find the active PR, pull its review comments, judge each one valid or a false positive, address it, and resolve the thread. Graduates to `engineering/`.
- **[fix-ci](./fix-ci/SKILL.md)** — Fetch CI status for the current PR or branch, read the failing run logs via the GitHub API, reproduce locally, fix, and push. Graduates to `engineering/`.
- **[translate-article](./translate-article/SKILL.md)** — Translate articles between Chinese and English, preserving structure, terminology, tone, and formatting. Graduates to `writing/`.
- **[translate-subtitle](./translate-subtitle/SKILL.md)** — Translate subtitle files (SRT, VTT, ASS) between Chinese and English, preserving timestamps and respecting line-length conventions. Graduates to `writing/`.
- **[paper-writing](./paper-writing/SKILL.md)** — Assist academic paper writing in Chinese or English — structure, section drafting, citations, venue style, and reviewer-style self-review. Graduates to `writing/`.
